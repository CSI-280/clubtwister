About Us

This is a web project created by Club Twister for our Open Source Software Development class, CSI-280

Our team: Steven Pershyn, Lenora Batcher, Jacob Buzzell, Jasmine Carr, Jared Remsberg

Our Goals

The purpose of our site is to advertise animals like merchandise in order to make it easy to find your perfect pet and find them the perfect home.

We think every pet deserves a forever home and this site is here to help accomplish that goal

How it Works

This site uses the petfinder API to provide a list of animals with helpful attributes for searching and displaying them back to the user. By using the petfinder API, we will be able to obtain pet objects with ID that can be added to a wishlist and retrieved at a later time.

This website is made with HTML, CSS, and Javascript
